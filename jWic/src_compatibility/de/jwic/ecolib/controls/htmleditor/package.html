WYSIWYG HTML editor control based upon the tinyMCE js library. The control is rendering a textarea and script code that forces the tinyMCE library to create a html editor.

To use this control, the tinyMCE library must be loaded and initialized in the main page. The following code must be included in your jwic.page file:

<SCRIPT LANGUAGE="JavaScript" SRC="$contextPath/jwic/ecolib/tiny_mce/tiny_mce.js"></SCRIPT>
<script language="javascript">
	tinyMCE.init({ theme: "simple" });	// initialize
</script>
