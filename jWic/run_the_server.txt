In order to run the WebServer to see the demos etc., you need to create a run configuration
in this project, using the main class:

	org.mortbay.start.Main
	
and the Arguments:
	
	jetty/jetty.xml
	
	
The server should then start up with the default configuration running at port 8080. Once it boot up, try
the following URL in your browser:

	http://localhost:8080/default/sample/
